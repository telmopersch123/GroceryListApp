## 📱 Sobre o Projeto

O **GroceryList** é um aplicativo mobile para organizar listas de compras de forma prática e intuitiva. Todas as suas listas ficam salvas localmente no seu celular, sem precisar de internet ou cadastro.

---

## ✨ Funcionalidades

- 📋 **Múltiplas listas** — Crie e gerencie quantas listas quiser
- 💾 **Armazenamento local** — Dados salvos no dispositivo com SQLite
- ⭐ **Favoritar listas** — Marque suas listas mais importantes
- 📋 **Copiar listas** — Duplique uma lista com todos os seus itens
- 🔗 **Compartilhar listas** — Envie sua lista para WhatsApp, Instagram e outros
- 🏷️ **Categorias** — Classifique suas listas em categorias personalizadas
- 🎨 **Tema claro/escuro** — Escolha o visual que preferir
- 🔔 **Notificações** — Sistema de lembretes para suas compras
- ✨ **Animações** — Interface fluida com Reanimated e Lottie
- 📊 **Progresso visual** — Acompanhe quantos itens já foram concluídos
